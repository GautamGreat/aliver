# Aliver

Aliver is Samsung Feature Phone [Unisoc Based] Flash Tool

It can flash Samsung Unisoc based devices

## Supported Models

```bash
Samsung E1200Y
Samsung E1207Y
Samsung B110E
Samsung B310E
Samsung B313E
```
